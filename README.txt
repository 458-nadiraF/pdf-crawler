PDF Web Crawler
===============

How to Run:
-----------
Method 1: Double-Click
1. Simply double-click "pdf-crawler.exe".
2. A window will open asking for a URL.
3. Paste the URL (right-click to paste in some terminals) and press Enter.

Method 2: Command Line
1. Open Command Prompt or PowerShell in this folder.
2. Type: pdf-crawler.exe
3. Press Enter.

Notes:
- The program will create a "downloads" folder in the same directory.
- It downloads up to 5 files at a time for speed.
- It is designed to be very fast (< 8 seconds for most pages).

Enjoy!
